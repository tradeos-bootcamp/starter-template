# TradeOS Bootcamp - Стартовый шаблон

Добро пожаловать в первый модуль bootcamp!

## Цель недели
Настроить окружение и создать первое работающее API.

## Задачи
1. Запустить среду разработки (через Codespaces или локально).
2. Дописать код в `app/main.py` по инструкции.
3. Протестировать API через Swagger UI.
4. Сделать Pull Request с выполненным заданием.

## Быстрый старт
1. Нажмите на зеленую кнопку `Code` -> `Open with Codespaces`.
2. После загрузки среды, в терминале выполните: `uvicorn app.main:app --reload`
3. Откройте в браузере: http://localhost:8000/docs
